#  Agrim Garg — Strategic Portfolio

> Systems Thinker • Builder • Future Civil Servant • Tech Strategist

An interactive, gamified personal portfolio designed to showcase not just skills — but mindset, growth, and long-term vision.

---

##  Vision

This portfolio is built as a strategic command center — blending:
- Gamified skill progression
- Leadership philosophy
- Interactive hiring experiences
- Long-term mission roadmap

It is not a resume website.  
It is a personal operating system.

---

##  Core Features

### 1. Skill Level Game Dashboard
- XP-based animated progress bars
- Level system (Lv. 1 – 10)
- Achievement badges (locked/unlocked states)
- Dynamic “Unlocked Skills” section
- Scroll-triggered animations

### 2. Personal Constitution
A dedicated section outlining 5 core principles that guide:
- Decision-making
- Leadership
- Discipline
- Long-term strategy
- Impact building

### 3. Interactive Hiring Suite
- “Hire Me → Choose My Next Skill” module
- Recruiter engagement cards
- Dynamic call-to-action section:
  > If you’re still scrolling, you should hire me.

### 4. Future Timeline
An animated roadmap from present to long-term goals:
- Academic milestones
- Startup initiatives
- Leadership growth
- Civil service mission

### 5. Growth & Impact Dashboard
- Public mission tracker
- Discipline heatmap
- Strategic thinking frameworks
- Founder-style decision logs

---

##  Design Philosophy

- Dark strategic dashboard aesthetic
- Minimal but powerful typography
- Gamified interactions
- High memorability factor
- Built for engagement, not decoration
